## 技术选型确认

- **前端**: React 18+、React Flow、Zustand、Ant Design、TypeScript
- **后端**: Python + FastAPI、LangGraph
- **数据库**: SQLite (开发阶段) / PostgreSQL (生产阶段)
- **实时通信**: WebSocket (后端推送运行状态)
